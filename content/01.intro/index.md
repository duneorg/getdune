## What is Dune?

Dune is a flat-file CMS for [Deno](https://deno.com). You write content in
Markdown, build themes with Preact TSX, and run the whole thing as a standard
Deno HTTP server. No database, no build pipeline, no vendor lock-in.

- [Deno](https://deno.com) 2.x or later
- A terminal and a text editor — that's it
